# Engagement Countdown

A simple, elegant web-based countdown timer created for our engagement celebration.

## 🚀 Features
* **Real-time Countdown:** Tracks the days, hours, minutes, and seconds until the big event.
* **Responsive Design:** Works on both desktop and mobile browsers.
* **Clean UI:** Minimalist HTML/CSS for a focused experience.

## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript
